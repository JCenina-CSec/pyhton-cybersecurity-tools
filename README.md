# Python Cybersecurity Tools

Collection of beginner cybersecurity-focused Python projects developed to practise networking, automation, file handling, reconnaissance, and security concepts.

## Included Projects

### Credential Manager
Password-protected credential storage application using SHA256 password hashing and ROT3-based credential obfuscation.

### TCP Port Scanner
Multithreaded TCP port scanner built using Python sockets and ThreadPoolExecutor for basic network reconnaissance and service discovery.
